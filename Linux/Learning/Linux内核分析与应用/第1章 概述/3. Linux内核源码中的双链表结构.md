# 链表的演化

在C语言中，一个基本的双向链表定义如下:

```c
struct my_list{
    void *mydata;
    struct my_list *next;
    struct my_list *prev;
};
```

这个链表画出图来的话是这样的, 它有前驱(prev)和后继(next), 这两个指针可以从两个方向来遍历链表
