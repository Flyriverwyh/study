
# 简介

前两节讨论了用于计时的时钟源：clocksource，以及内核内部时间的一些表示方法，但是对于真实的用户来说，我们感知的是真实世界的真实时间，也就是所谓的墙上时间，clocksource只能提供一个按给定频率不停递增的周期计数，如何把它和真实的墙上时间相关联？本节的内容正是要讨论这一点。

# 时间的种类

内核管理着多种时间，它们分别是：

RTC时间
wall time：墙上时间
monotonic time
raw monotonic time
boot time：总启动时间
