# 方式一: 增加磁盘数

## VMware Fusion设置

关闭虚拟机后, 增加磁盘

"虚拟机" "设置" "添加设备" "新硬盘"

![2019-09-13-22-15-48.png](./images/2019-09-13-22-15-48.png)

![2019-09-13-22-16-21.png](./images/2019-09-13-22-16-21.png)

设置好需要添加的硬盘信息，点击【应用】添加磁盘。

至此, VMware Fusion设置完成
