在保护模式下，通过将内存分成大小不等的段，并用描述符对每个段的用途、类型和长度进行指定，就可以在程序运行时由处理器硬件施加访问保护。

段保护时处理器提供的基本保护功能，但对于现实的需求来说，仍是不够的。
